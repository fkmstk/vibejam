# vibejam

`月下ノ刃` の雰囲気プロトタイプ。

白い和紙をベースに、黒い水墨と赤い鳥居をアクセントにした、
一太刀カウンター忍者ゲームの Web デモ。

## Play

- Live: [https://vibejam-mu.vercel.app](https://vibejam-mu.vercel.app)
- Repo: [https://github.com/fkmstk/vibejam](https://github.com/fkmstk/vibejam)

## Concept

- 直感でわかる 1 アクション
- 水墨画っぽい白基調の画面
- 赤い予兆を見て斬る、短いプレイサイクル
- Vibe Coding Game Jam 向けの軽量な Web 構成

## Stack

- TypeScript
- Vite
- Vanilla DOM / CSS
- Vercel

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

GitHub の `master` へ push して、Vercel の `vibejam` project に接続済み。
